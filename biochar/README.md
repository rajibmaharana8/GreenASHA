# GreenASHA AI Sustainability Platform

Professional MRV (Monitoring, Reporting, and Verification) landing page and backend system.

## Project Structure
- **/frontend**: React + Vite landing page with high-end aesthetics.
- **/backend**: Node.js + Express API for ROI calculations and contact inquiries.

## How to Run

### 1. Start the Backend
```bash
cd backend
npm install
npm start
```
The server will run on `http://localhost:5000`.

### 2. Start the Frontend
```bash
cd frontend
npm install
npm run dev
```
The landing page will be available on `http://localhost:5173`.

## Key Features
- **Modern Landing Page**: Built with React and high-end Vanilla CSS.
- **Glassmorphism UI**: Premium visual style with smooth transitions.
- **Interactive ROI Calculator**: Estimate financial and environmental impact.
- **Digital MRV Integration**: Built-in dashboard previews for carbon tracking.
- **Responsive Design**: Fully optimized for mobile and tablet devices.
